# 易考 AI 自动化初版

本项目用于根据易考考试需求单自动完成后台考试配置。

## 主要能力

- 导入 Excel 需求单并解析考试名称、考试时间、提前登录、限制迟到、欢迎语、考前等待提示、考试承诺书、考试配置项和科目信息。
- 通过本地网页显示导入结果、执行进度、执行日志和确认截图。
- 使用 Playwright 接管易考后台，自动完成新建考试流程。
- 支持高级编辑器的“源代码”模式写入并保存富文本内容。

## 本地启动

```bash
node server/easy_exam_server.mjs
```

访问：

```text
http://127.0.0.1:8765
```

## 说明

运行时上传文件、截图和本地登录配置保存在 `.easy_exam_runtime/`，该目录不会提交到 GitHub。
